Typography
We’ll be using just 1 font face for our entire web app. The reason behind this is that we really want our web app to be consistent in all the aspects.
Montserrat
This will be used for all the text on the web app except for the logo. We chose Montserrat because of it’s subtle, clean and minimalistic look and feel.
